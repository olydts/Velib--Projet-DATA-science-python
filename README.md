# Velib--Projet-DATA-science-python
Projet ayant pour but d'optimiser la régulation des stations Vélib' en analysant les flux, les déséquilibres et les interventions.
## Méthodologie
- Préparation et nettoyage des données
- Analyse exploratoire
- Analyse des régulations
- Clustering K-Means
- Pistes de modélisation prédictive

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib / Plotly
